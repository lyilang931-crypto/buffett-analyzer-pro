# Moat — 価値投資分析プラットフォーム

7つのモート原則で全世界の株式をリアルタイム分析・スクリーニング。

## 特徴

- **7つのモート原則** — 能力の輪・経済的堀・安全マージン・長期耐久性・優秀経営・Mr.Market・顧客愛
- **グローバルスクリーニング** — 日米欧アジア26銘柄以上をリアルタイム分析
- **CEO評価** — バフェット目線＋起業家目線の二軸スコアリング
- **市場評価指数** — 米国市場の過熱度をリアルタイム表示
- **10倍株確率** — モートスコアに基づく10倍株候補の確率算出
- **ウォッチリスト** — 銘柄の追加・削除をローカルに保存

## 技術スタック

- Next.js 14 (App Router)
- Yahoo Finance API (`yahoo-finance2`)
- Tailwind CSS (ブラック・ゴールドテーマ)
- TypeScript
- Vercel

## 開発環境

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) で確認。
